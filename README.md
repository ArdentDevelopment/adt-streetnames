# adt-streetnames

### **Features**

* **4 Adjustable Themes:** Choose from four distinct visual styles to match your server's aesthetic.
* **Flexible Positioning:** Move the UI element to any part of the screen with a simple position configuration.
* **Toggleable Map Icon:** Enable or disable the map icon next to the street names based on your preference.
* **Customizable Refresh Rate:** Optimize performance by adjusting the control intervals (check times) to your liking.
